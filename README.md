⚡ LastMinuteAI

> **Your AI-powered deadline companion** — plan smarter, prioritize better, and rescue any task before the clock runs out.

---

## 📌 Problem Statement

Deadlines are brutal. Whether you're a student juggling assignments, a developer racing to ship, or a professional managing back-to-back deliverables — poor planning and unclear priorities cost valuable time every day.

Most people know *what* needs to be done. They just don't know **where to start**, **what to drop**, or **how to make it fit in the time they have left**.

LastMinuteAI solves this with four AI agents that give you a personalized action plan in seconds.

---

## 🚀 Features

### ▦ Task Board
Track all your commitments in one place. Add tasks with deadlines and estimated hours — the board automatically flags urgent items, calculates total workload, and shows a live focus score.

### ⬡ AI Planner
Describe any task in plain English and get a numbered, time-estimated, step-by-step action plan generated by Gemini AI. No vague advice — just concrete next steps.

### ↑↑ Prioritize
Paste a list of tasks (one per line) and let the AI rank them by urgency and impact. Each task comes back with a `HIGH / MEDIUM / LOW` urgency badge and a brief reason, so you know exactly what to do first.

### ⚡ Rescue Mode
Running out of time? Enter your task, how many hours you need, and how many you actually have. The AI returns an emergency plan with what to focus on, what to drop, a compressed schedule, pro tips, and a success probability estimate.

### ◎ Daily Coach
Give the AI your full task list for the day. It returns a motivational goal, a recommended hour-by-hour schedule, quick wins to build momentum, distractions to avoid, and your completion probability.

## 🧠 How It Works

```
User Input
    │
    ▼
React Frontend  ──────────►  FastAPI Backend
(Vite + React 18)              (Python 3.x)
                                    │
                          ┌─────────┴──────────┐
                          │   AI Agent Layer   │
                          │ ┌────────────────┐ │
                          │ │ Planning Agent │ │
                          │ │ Priority Agent │ │
                          │ │  Rescue Agent  │ │
                          │ │  Coach Agent   │ │
                          │ └────────────────┘ │
                          └─────────┬──────────┘
                                    │
                          ┌─────────▼──────────┐
                          │   Gemini Service   │
                          │  gemini-2.5-flash  │
                          └────────────────────┘
```

Each AI agent is a standalone Python module with a targeted prompt. The Gemini service wraps `google-generativeai` and returns either plain text (for the Planner) or structured JSON (for Priority, Rescue, and Coach), parsed and validated before reaching the frontend.

## ⚙️ Getting Started

### Prerequisites

- Python 3.10+
- Node.js 18+
- A [Google Gemini API key](https://aistudio.google.com/app/apikey)

---

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
```

Create a `.env` file in the `backend/` directory:

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

Start the server:

```bash
uvicorn main:app --reload
```

The API will be running at `http://localhost:8000`.

---

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The app will be running at `http://localhost:5173`.

## 📡 API Reference

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/` | Health check |
| `POST` | `/plan` | Generate a step-by-step action plan |
| `POST` | `/priority` | Rank a list of tasks by urgency |
| `POST` | `/rescue` | Get an emergency plan for tight deadlines |
| `POST` | `/daily-coach` | Build a full daily schedule and coach plan |

### Example — Rescue Mode

**Request**
```json
POST /rescue
{
  "task": "Machine learning project report",
  "hours_needed": 8,
  "hours_left": 4
}
```

**Response**
```json
{
  "risk": "HIGH",
  "summary": "You have half the time you need — ruthless prioritization is essential.",
  "focus": ["Core model training and writeup"],
  "postpone": ["Hyperparameter tuning", "Extended literature review"],
  "schedule": ["Now–2PM: Data pipeline and model run", "2PM–4PM: Results writeup and conclusion"],
  "tips": ["Use a pre-trained baseline to save 2h", "Write while the model trains"],
  "success_probability": "70%"
}
```
